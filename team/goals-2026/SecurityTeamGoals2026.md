# Security Team Goals - FY2026

*For Observability team members seconded to Security team for Runtime Enforcer, Network Scanner, and Security Hub integration work.*

---

## Goal 1: Runtime Enforcer Delivery

**Goal:** Contribute to the delivery of Runtime Enforcer GA by KubeCon Europe (April 2026).

**Key Results:**
- Complete assigned development tasks for Runtime Enforcer feature set
- Ensure code quality through comprehensive testing and code review participation
- Support integration testing and bug resolution during stabilization phase
- Collaborate with Security team on documentation and release readiness

**Success Metric:** Runtime Enforcer reaches GA quality with assigned components delivered on schedule

---

## Goal 2: Network Scanner Delivery

**Goal:** Contribute to Network Scanner development, achieving Experimental release by August 2026 and GA by KubeCon NA (November 2026).

**Key Results:**
- Deliver assigned Network Scanner components according to sprint commitments
- Participate in architecture discussions and technical design reviews
- Support testing and validation of network scanning capabilities
- Ensure AppCo delivery readiness for GA release

**Success Metric:** Network Scanner milestones met (Exp: August, GA: November) with quality deliverables

---

## Goal 3: Security Hub Integration

**Goal:** Integrate Runtime Enforcer and Network Scanner with Security Hub by end of FY2026 (January 2027).

**Key Results:**
- Implement data ingestion pipelines from Runtime Enforcer into Security Hub
- Enable Network Scanner visualization and alerting within Security Hub UI
- Ensure seamless user experience across integrated security components
- Complete integration testing and documentation for end-to-end workflows

**Success Metric:** Both Runtime Enforcer and Network Scanner fully operational within Security Hub by EOY FY2026
